# Day 2 | UI/UX
```
    1. Design thinking , Empathize -> Define -> Ideate -> Prototype -> Validate
    
    2. At Designing the UX , first we need to define the user persona, after that we make the stories 
    based on our user persona then we check the performance through usability testing step with survey. 
    after that we move on to the define process where we got the data from the empathize stage, we need 
    to define what's the most important for our user and our product after that we make the user flow 
    using wireflow that indicates the step from beginning until the end.
    
    3. For the UI, there's 4 pillar of UI , Consistency , Simple Interface , Good Typography and Informative Feedback. 
    This 4 pillar help us to design the best Interface that we can offer to the user
```
